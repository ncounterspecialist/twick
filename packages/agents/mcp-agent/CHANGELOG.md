# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-XX

### Added
- Initial release of MCP Twick Server
- Video transcription using Google Vertex AI (Gemini)
- Multi-language and font support
- Integration with Twick Studio
- One-click installation script for Claude Desktop
- Support for downloadable project JSON files
- Automatic path resolution for GCP credentials
- Cross-platform support (Windows, macOS, Linux)

### Features
- `generate-subtitles` tool for Claude Desktop
- Automatic subtitle generation with timing
- Project file upload to S3 (optional)
- Twick Studio link generation with `$project` placeholder support
- Comprehensive error handling and logging
